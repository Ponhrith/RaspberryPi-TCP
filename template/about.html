Summary

This project was a challenge raised by teacher Solyka to solve manual watering problem in the school plant's net house. With this opportunity to apply what we have learnt in our Network Programming class, our mentor, teacher Yuki guided us to work on the project.The project uses Python as the programming language and TCP/IP as a protocol. For the software part, the program is divided into two parts, client and server. Our laptop acts as a server, waiting a connection from RaspberryPi which acts as a client that reads data from the sensors, write the data into a text file and send it to the server.

Components

RaspberryPi Zero W

DHT11 Humidity Sensor

DC Water Pump Motor

Soil Moisture Sensor
